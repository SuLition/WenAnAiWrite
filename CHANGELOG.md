# 更新日志

## v0.2.9

### 新功能

- **关闭最小化到托盘**：点击关闭按钮时可选择最小化到系统托盘
- **外观设置页面**：新增卡片动画、动画速率等个性化配置项
- **缓存管理功能**：支持音视频缓存文件夹大小统计和一键清理
- **任务队列增强**：支持添加本地音频与文案任务，支持视频下载任务
- **平台解析统一接口**：新增平台检测与视频解析的统一入口
- **音频播放器组件**：集成复用音频播放器组件

### 修复

- 修复列表删除卡片时，其他卡片补位没有位移动画的问题
- 修复动画持续时间最小值限制问题
- 修复腾讯 ASR 返回格式，统一返回纯文本并清理时间戳前缀

### 优化

- 优化卡片动画曲线，统一使用弹性贝塞尔曲线，动画更流畅
- 优化界面圆角及解析按钮动画效果
- 调整动画速率配置和过渡样式范围
- 优化默认窗口效果及移除加载动画
- 重构后端模块代码结构
- 重构任务队列功能和界面交互

## v0.2.0

### 新功能

- **应用内更新**：支持自动检查更新，可在设置中开启/关闭
- **主题色自定义**：新增 7 种主题色可选（天空蓝、翡翠绿、薰衣紫、琥珀橙、玫瑰粉、青绿色、珊瑚红）
- **视频 ID 去重**：自动过滤重复的历史记录
- **历史记录上限设置**：可自定义最大保存数量
- **链接自动检测**：解析时自动识别平台，无需手动切换
- **输入框清空按钮**：一键清空输入内容

### 优化

- **更新弹窗优化**：调整尺寸，新增下载进度显示
- **加载动画改进**：新增公共加载按钮样式、进度按钮样式
- **仓鼠跑轮动画**：解析等待页面使用全新可爱动画
- **历史记录删除按钮**：改为图标样式，更简洁
- **设置图标**：修复选中状态，更换为线性齿轮风格
- **代码重构**：ParsePage 拆分为模块化组件，便于维护

### 移除

- 移除快手平台支持
